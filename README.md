# BrainfuckToC
Program that compiles .b and .bf files into .c files. Written in Python

# How To Use
1.  Paste bf_to_c into the folder with your code
2.  Run ```bf_to_c <yourfilename>.bf```
3.  Your file will be saved as ```<yourfilename>.c```
